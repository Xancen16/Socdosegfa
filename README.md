# skill-of-checking-discounts-on-Steam-Epic-Games-for-Alisa (SOCDOSEGFA)
Суть проекта:

Интеллектуальный агрегатор игровых скидок, который помогает пользователю не пропускать выгодные предложения и бесплатные раздачи игр в крупнейших цифровых магазинах (Steam и Epic Games Store), не выходя из голосового интерфейса Алисы.

Что умеет навык
Мониторинг Steam: В реальном времени запрашивает через API список игр с самыми большими скидками на данный момент.
Отслеживание раздач в EGS: Находит игры с ценой 0 рублей в Epic Games Store, сообщая пользователю о возможности забрать игру бесплатно.
Память и персонализация: Благодаря интеграции с базой данных (ORM), навык узнает пользователя, помнит его имя и последний магазин, которым он интересовался.
